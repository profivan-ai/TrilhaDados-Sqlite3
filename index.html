Vídeos da Playlist

Exercícios sugeridos

  • Crie uma tabela de usuários no Supabase com campos nome, email e idade.
  • Desenvolva uma página HTML que conecte e exiba os dados dessa tabela.
  • Implemente uma funcionalidade de cadastro de novos usuários usando o Supabase.
  • Adicione autenticação de login e logout com o sistema do Supabase.
  • Crie uma interface de edição e exclusão dos usuários cadastrados.
